# **[Opposite number](https://www.codewars.com/kata/56dec885c54a926dcd001095/javascript)**

## **Description**:

Very simple, given an integer or a floating-point number, find its opposite.

## **Examples**:

```
(input --> output)
1: -1
14: -14
-34: 34
```

## **Solution**:

```javascript
const opposite = (number) => -number;
```
